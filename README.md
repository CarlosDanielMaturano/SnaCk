# SnaCk
A terminal based simple snake game

### Dependencies: :memo:
- make
- clang

### Building the project: :hammer:
```bash
make build
```

### Running the project: :rocket:
```bash
make run
```

### Building & Running project: :shipit:
```bash
make all
```
